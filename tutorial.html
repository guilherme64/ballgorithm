Elementos

O usuário deve resolver os seguintes desafios da seguinte forma: Criar os canos existentes no sistema através destes comandos:

Criar bola:

Criar uma bola com um valor inteiro

new ball = 5;

Cano de Atribuição:

Cano atribuição


pipe is [operação básica] [número inteiro];

exemplo:
pipe is + 3;

Cano de estrutura de condicional (if):

Cano estrutura condicional



if ball [== ou !=] [número inteiro]:
 *indentação de dois espaços* [condição caso seja verdadeiro( cano esquerdo do if)]
else:
 *indentação de dois espaços* [condição caso seja verdadeiro( cano esquerdo do if)]
endif;
exemplo:
if ball == 4:
 pipe is + 4;
else:
 pipe is -3;
endif;

Cano de estruturas de repetição:

Cano de estruturas de repetição

while ball [== ou !=] [número inteiro]:
 *indentação de dois espaços* [condição caso seja verdadeiro( cano esquerdo do if)]
endif;

exemplo:
while ball != 3:
 pipe is + 3;
endwhile;

Desafios

  1. Criar uma bola de valor 2 e alterar o seu valor para 4
  2. Criar duas bolas com valor 1 e 3 e retornar duas bolas de valor 2
  3. Criar uma bola de valor um e, através de estruturas de repetição, retornar o valor 5
